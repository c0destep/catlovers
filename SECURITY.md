# Política de Segurança

## Versões Suportadas

Atualmente, apenas a versão principal mais recente recebe atualizações de segurança.

| Versão | Suportada          |
|:-------|:-------------------|
| 1.x    | :white_check_mark: |
| < 1.0  | :x:                |

## Relatando uma Vulnerabilidade

A segurança deste projeto é levada a sério. Se você encontrar uma vulnerabilidade de segurança, por favor, **não abra
uma Issue pública**. Em vez disso, relate o problema de forma privada para que possamos resolvê-lo adequadamente.

Para relatar uma vulnerabilidade, envie um e-mail para:
**lucas.alves.capristrano59@gmail.com**

### O que incluir no relato:

- Descrição detalhada da vulnerabilidade.
- Passos para reproduzir o problema (PoC).
- Impacto potencial que a vulnerabilidade pode causar.

Agradecemos por nos ajudar a manter o **Catlovers** seguro para todos os usuários e colaboradores.
