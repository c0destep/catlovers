# Estudo: API de Adoção

Este documento serve como ponto de partida para o estudo e implementação da API de Adoção para o projeto Catlovers.

## Objetivo

Fornecer uma interface programática para consulta de gatos disponíveis para adoção, integrando protetores e famílias de
forma tecnológica.

## Status: WIP (Work In Progress)

- [ ] Definição da arquitetura (REST/GraphQL).
- [ ] Escolha da stack (mantendo foco em tecnologias fundamentais).
- [ ] Modelagem de dados (Gatos, ONGs, Usuários).
- [ ] Implementação de endpoints básicos.

## Observações

O desenvolvimento desta API está em fase de estudo e não possui previsão de lançamento imediato.
